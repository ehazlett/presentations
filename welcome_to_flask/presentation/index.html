Welcome to Flask
What is it?
Flask is a Python based microframework.

* Dead simple to use

* Simple url routing

* Templates via Jinja2 by default

* Built-in debugger

* Extensions

* much more :)

Diving right in... 
        
        from flask import Flask
        app = Flask(__name__)

        @app.route("/")
        def hello():
          return "Hello World!"

        if __name__ == "__main__":
          app.run()
Decorator based routing 
      
        @app.route("/about", methods=['GET', 'POST'])
        def about():
          return render_template('about.html')
...with variable rules 
      
        @app.route("/accounts/<username>/")
        def user_account(username=None):
          ...

        @app.route("/accounts/<int:user_id>/")
        def user_account(user_id=None):
          ...
URL Building 
      
        from flask import Flask, url_for, redirect
        app = Flask(__name__)

        @app.route("/")
        def index():
          return "Hello world"
        
        @app.route("/login")
        def login():
          login()
          redirect(url_for('index'))
Templates 
      
        from flask import render_template

        @app.route("/about")
        def about():
          version = 'Release 0.1'
          return render_template('about.html', version=version)
...and the corresponding template 
      
        <!doctype html>
        <title>about</title>

        {% if version %}
        <p>Version: {{version}}</p>

        {% else %}
        <p>Version: unknown</p>

        {% endif %}
Logging

Comes with built-in Python logger

      
        from flask import Flask
        app = Flask(__name__)
        
        app.logger.debug('Bootstrap...')
        app.logger.error('Error occurred')
